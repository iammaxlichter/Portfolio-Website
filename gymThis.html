GymThis


This is a side project web app my friend and fellow classmate, Jahnzeb Bastaki and I are in the early stages of creating. We plan for this project to be done by the start of Summer 2023.

Do you ever feel like there is too much things to remember in the gym? How many reps/sets, what each exercise does, and the progress you need to be making has all been integrated into GymThis. GymThis allows new and experienced gym goers alike to help store principal information regarding the gym.
